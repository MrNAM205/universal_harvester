# OmniVeroBrix Cockpit Global Intelligence — Specification (v1.0)

Global Intelligence is composed of seven intelligence organs, forming the unified brain of OmniVeroBrix. It synthesizes all domains into a coherent worldview.

## 1. Global Entity Intelligence
Manages the unified entity universe across all domains.
- **Responsibilities:** extraction, normalization, deduplication, linking, clustering, confidence scoring.
- **Outputs:** entity graph, clusters, issuer families, recurring parties. Access via Knowledge Graph.

## 2. Global Corpus Intelligence
Manages the legal, regulatory, procedural, and sovereign knowledge base.
- **Responsibilities:** ingestion, indexing, procedural mapping, cross-referencing, remedy map generation.
- **Outputs:** corpus map, procedural/domain clusters, remedy maps. Access via Corpus Panel.

## 3. Global Remittance Intelligence
Manages the financial instrument universe.
- **Responsibilities:** detection, coupon parsing, account/routing clustering, issuer pattern detection, timeline extraction.
- **Outputs:** remittance map, issuer/account clusters, remittance geometry patterns. Access via Remittance Panel.

## 4. Global Mission Intelligence
Manages the mission universe.
- **Responsibilities:** lifecycle, dependency graph, scheduling, routing, orchestration, intelligence sharing.
- **Outputs:** dependency graph, cluster map, orchestration plan, intelligence ledger. Access via Mission Orchestration Panel.

## 5. Global Autonomy Intelligence
Governs global autonomous reasoning across all missions.
- **Responsibilities:** cross-mission autonomy/reflections/decisions, safety boundaries, stop conditions.
- **Outputs:** autonomy ledger, decision/reflection clusters, global timeline. Access via Autonomy Monitor.

## 6. Global Memory Intelligence
Manages long-term sovereign memory across the entire cockpit.
- **Responsibilities:** operator, mission, document, corpus, remittance, autonomy memory.
- **Outputs:** memory graph, clusters, timeline, stability report. Access via Memory Graph.

## 7. Global Situational Intelligence
Synthesizes everything into a single situational model.
- **Responsibilities:** cross-domain intelligence fusion, global timeline/cluster/anomaly detection, global summaries.
- **Outputs:** global intelligence map, timeline, cluster universe, anomaly report, summary. Access via Global Dashboard.
