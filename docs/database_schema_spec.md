# OmniVeroBrix Database Schema (v1.0)
*(SQLite‑optimized, relational, sovereign‑grade)*

## 📄 1. harvested_documents
Stores every document ingested by the Harvester.
| Column | Type | Description |
|-------|------|-------------|
| id | TEXT (UUID) | Primary key |
| source | TEXT | pdf, image, html, email, chat |
| ingested_at | TEXT | ISO8601 |
| metadata_json | TEXT | Raw metadata block |
| classification_json | TEXT | Document type + tags |
| parties_json | TEXT | Sender/recipient/others |
| dates_json | TEXT | All extracted dates |
| financials_json | TEXT | Amounts, line items |
| remittance_json | TEXT | Coupon/routing/etc |
| entities_json | TEXT | Extracted entities |
| text_blocks_json | TEXT | Page‑level text |
| full_text | TEXT | Entire document text |
| timeline_json | TEXT | Extracted timeline |
| next_actions_json | TEXT | Suggested next steps |

## 📚 2. corpus_entries
Stores CFR, USC, UCC, IRM, agency guidance, sovereign archives.
| Column | Type | Description |
|--------|------|-------------|
| entry_id | TEXT (UUID) | Primary key |
| type | TEXT | statute, regulation, manual, guidance, maxim, archive |
| jurisdiction | TEXT | federal, state, agency |
| citation | TEXT | e.g., 12 CFR 1026.5 |
| title | TEXT | Section title |
| body | TEXT | Full text |
| tags_json | TEXT | Domain tags |
| related_entries_json | TEXT | Cross‑references |
| source | TEXT | CFR, USC, UCC, IRM, Archive |
| ingested_at | TEXT | ISO8601 |

## 🔗 3. document_corpus_mappings
Links harvested documents to corpus entries.
| Column | Type | Description |
|--------|------|-------------|
| mapping_id | TEXT (UUID) | Primary key |
| document_id | TEXT | FK → harvested_documents.id |
| corpus_entries_json | TEXT | List of entry_ids + reasons |
| derived_context_json | TEXT | Process type, roles, terms |

## 🧭 4. remedy_maps
Stores structural remedy maps generated by the Corpus Layer.
| Column | Type | Description |
|--------|------|-------------|
| remedy_map_id | TEXT (UUID) | Primary key |
| document_id | TEXT | FK |
| summary | TEXT | High‑level explanation |
| process_json | TEXT | Name + description |
| deadlines_json | TEXT | Extracted deadlines |
| possible_paths_json | TEXT | Structural options |
| notes | TEXT | Additional context |

## 🎯 5. missions
Stores missions created by the Mission Router.
| Column | Type | Description |
|--------|------|-------------|
| mission_id | TEXT (UUID) | Primary key |
| mission_type | TEXT | analysis, ingestion, corpus, remittance, etc. |
| status | TEXT | pending, running, blocked, complete |
| created_at | TEXT | ISO8601 |
| updated_at | TEXT | ISO8601 |
| inputs_json | TEXT | Raw user input + parameters |
| requirements_json | TEXT | Missing fields, clarifications |
| assigned_to_json | TEXT | Subsystems |
| result_json | TEXT | Final output |

## 🧩 6. mission_steps
Stores step‑level execution inside missions.
| Column | Type | Description |
|--------|------|-------------|
| step_id | TEXT (UUID) | Primary key |
| mission_id | TEXT | FK → missions.mission_id |
| description | TEXT | Step description |
| subsystem | TEXT | assistant, harvester, corpus, remittance |
| status | TEXT | pending, running, complete |
| result_json | TEXT | Output of the step |

## 🧠 7. thought_ledger
Stores internal reasoning entries (never shown to user).
| Column | Type | Description |
|--------|------|-------------|
| entry_id | TEXT (UUID) | Primary key |
| mission_id | TEXT | FK |
| timestamp | TEXT | ISO8601 |
| type | TEXT | observation, hypothesis, plan, reflection, decision |
| content | TEXT | Internal reasoning summary |
| metadata_json | TEXT | confidence, related entities |

## 🧪 8. plans
Stores multi‑step plans generated by the Sovereign Thought Engine.
| Column | Type | Description |
|--------|------|-------------|
| plan_id | TEXT (UUID) | Primary key |
| mission_id | TEXT | FK |
| steps_json | TEXT | Ordered plan steps |

## 💳 9. remittance_instruments
Stores parsed remittance coupons.
| Column | Type | Description |
|--------|------|-------------|
| instrument_id | TEXT (UUID) | Primary key |
| document_id | TEXT | FK |
| instrument_type | TEXT | utility_bill_coupon, credit_card_stub, etc. |
| issuer_name | TEXT | Company/agency |
| issuer_reference | TEXT | Reference number |
| account_number | TEXT | Parsed account number |
| routing_number | TEXT | Parsed routing number |
| coupon_code | TEXT | Parsed coupon code |
| payment_address | TEXT | Parsed remit‑to address |
| payment_methods_json | TEXT | Allowed payment methods |
| due_date | TEXT | ISO8601 |
| amount_due | REAL | Numeric |
| barcode_data | TEXT | Raw barcode |
| scanline | TEXT | Raw scanline |

| Column | Type | Description |

| timestamp | TEXT | ISO8601 |
| level | TEXT | info, warning, error |

| metadata_json | TEXT | Additional context |
